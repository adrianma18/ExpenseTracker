# Expense Tracker

CRUD React - Tailwind - Firebase app to keep track of personal expenses.

## Authors

- [@adrianma18](https://github.com/adrianma18)

## Run Locally

Clone the project

```bash
  git clone https://link-to-project
```

Go to the project directory

```bash
  cd my-project
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run start
```
